# Portfólio — Manuella Maia

Portfólio profissional de desenvolvimento web, criado para apresentar projetos reais, habilidades técnicas e o processo de trabalho. O site foi construído do zero com React e CSS puro — sem UI kits — priorizando performance, responsividade e identidade visual própria.

## Tecnologias

- **React + Vite** — estrutura de componentes e build otimizado
- **CSS puro** — estilização completamente customizada, sem frameworks
- **JavaScript** — animações, scroll reveal e interações

## Seções

| Seção | Descrição |
|---|---|
| Hero | Apresentação com typing animation e cards flutuantes com parallax |
| Projetos | Portfólio interativo com navegação por abas e preview de cada projeto |
| Experiência | Habilidades técnicas, soft skills e tecnologias utilizadas |
| Como Funciona | Processo de contratação em 4 etapas compactas |
| Depoimentos | Feedbacks reais de clientes com carrossel automático |
| FAQ | Perguntas frequentes com acordeão animado |
| Contato | Modal de contato e links diretos |

## Rodando localmente

```bash
npm install
npm run dev
```

## Build para produção

```bash
npm run build
```
