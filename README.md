# Portfólio — Manuella Maia

Site de portfólio pessoal desenvolvido para apresentar projetos, habilidades e serviços de desenvolvimento web.

## Tecnologias

- React + Vite
- CSS customizado (sem frameworks)
- JavaScript

## Seções

- **Hero** — apresentação com typing animation e cards flutuantes
- **Projetos** — portfólio com navegação por abas
- **Experiência** — habilidades técnicas e soft skills
- **Como Funciona** — processo de contratação em 4 etapas
- **Depoimentos** — feedbacks de clientes
- **FAQ** — perguntas frequentes
- **Contato** — formulário e links diretos

## Como rodar localmente

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
```
